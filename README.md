# Bala Prabhakar B DevOps Resume Website

## Deployment Steps for Nginx

```bash
sudo apt update
sudo apt install nginx -y
sudo unzip bala_prabhakar_devops_resume.zip
sudo cp -r * /var/www/html/
sudo systemctl restart nginx
```
